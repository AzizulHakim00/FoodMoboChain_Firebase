# FoodMoboChain v1.3.0

## Added

- Multi-store marketplace directory and storefronts
- Large repeatable starter database
- Promotional campaigns and discounted catalogue
- Saved delivery addresses and delivery notes
- Support-ticket conversations
- Vendor and administrator analytics
- Stock-aware menu inventory and carts
- Store-split orders
- Packed order stage
- Expanded Security Rules and emulator coverage

## Changed

- Official food `price` is always the value validated and charged at checkout.
- `regularPrice` is used only to show the pre-discount comparison.
- Approved vendors receive a storefront and manage stock through Seller Centre.
- Existing foods without the v1.3 stock field remain visible for upgrade compatibility.
